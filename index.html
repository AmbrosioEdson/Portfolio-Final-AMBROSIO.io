Reflexión acerca de Desarrollo de Aplicaciones Web

En este curso me ha preparado para abordar proyectos web de manera integral, desde la interfaz de usuario hasta el backend, con conocimientos sólidos en tecnologías actuales y habilidades prácticas para enfrentar desafíos del mundo real. he adquirido una sólida comprensión de las tecnologías fundamentales, desde los principios de la web hasta los protocolos como TCP/IP y HTTP. He construido páginas web utilizando HTML y XML, aprendí a diseñar con CSS3 y utilicé librerías como Bootstrap y Tailwind CSS. Trabajé con JavaScript para manipular el DOM y crear gráficos con Canvas.Exploré frameworks JS para el desarrollo frontend, comprendí Next.js para renderizado del lado del cliente y servidor, así como generación estática e incremental. Además, me sumergí en el desarrollo backend con Java y Spring Boot, construyendo API REST.
